## Beginner 
- SELECT Statment - used to query a database and retrieve data from one or more tables
- WHERE clause - used to filter records and extract only the rows that fulfill a specific condition
- GROUP BY - clause arranges rows with identical data into summary groups
- HAVING vs WHERE clauses
  a - HAVING clause - is a filtering tool used to filter the results of a query after rows have been grouped by a GROUP BY clause
  b - WHERE clause - used to filter records and extract only the rows that fulfill a specific condition
-   LIMIT & ALIAS
  a - LIMIT -  used to restrict the maximum number of rows that a query return
  b - ALIAS - is used as a temporary name assigned to a table or a column within a query to make the code cleaner, shorter, and easier to read

## Intermediate
- JOINS - is a command clause used to combine rows from two or more database tables based on a related column between them
- UNION - is an operator used to combine the result sets of two or more SELECT queries into a single, unified result set
- STRING functions - built-in scalar tools used to manipulate, format, and transform text data within database management systems
- CASE statements - statements evaluates conditional logic within a query and returns a specific value as soon as a condition is met
- WINDOW functions -  performs a calculation across a set of table rows that are related to the current row
